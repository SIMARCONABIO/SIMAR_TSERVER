# SIMAR_TSERVER
SIMAR Flask TileServer

```
pip3 install Pillow=2.2.2
pip3 install MapProxy
pip3 install webtest
pip3 install PyYAML
```

### Create Virtual environment
```
python3 -m venv /data/apps/simar_tileserver/venv
source venv/bin/activate
deactivate
```
